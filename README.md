## Overview
This project provides a technical framework to simulate how various hypothetical portfolio structures would have performed if subjected to the specific economic conditions of the Japanese "Lost Decades." 

By using the 1990 Japanese market peak as a starting point, the tool allows researchers to observe the impact of prolonged stagnation, deflation, and low interest rates on various retirement withdrawal strategies.

## Technical Scope
* **Historical Data Integration:** Models the Nikkei 225 and Japanese bond market conditions from 1990 through the subsequent decades.
* **Portfolio Robustness Analysis:** Provides a platform for testing "weatherproof" theories against actual historical data points rather than randomized probabilities.
* **Sequence of Returns Research:** Specifically examines the mathematical impact of a major market peak followed by decades of sideways or downward movement.

## Simulation Variables
Users can configure the following parameters within the simulation scripts to test the resiliency of the logic:
* **Hypothetical Allocations:** Test different ratios of domestic versus international assets.
* **Withdrawal Logic:** Apply dynamic spending rules (e.g., "Guardrails") to observe how they would have mathematically mitigated capital depletion.
* **Inflation/Deflation Adjustments:** Toggle between various purchasing power assumptions based on historical Japanese CPI data.
